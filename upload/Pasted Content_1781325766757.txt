2.4 — NAVIGATION PATTERNS NATIVOS

TRANSICIONES ENTRE PANTALLAS:
- Avanzar (de lista a detalle): la nueva pantalla slide-in desde la derecha, la vieja se desliza levemente a la izquierda (parallax)
  translateX(100%) → translateX(0) para la nueva
  translateX(0) → translateX(-30%) para la vieja
  Duracion: 0.35s, cubic-bezier(0.2, 0, 0, 1)
  
- Volver atras: reversa exacta de la animacion anterior
  Swipe desde el borde izquierdo: gesture que arrastra la pantalla actual a la derecha revelando la anterior
  Si el swipe supera 40% del ancho: completa la transicion. Si no: vuelve.

- Subir (a bottom sheet): translateY(100%) → translateY(0), 0.4s spring
- Bajar (cerrar sheet): reversa

- Fade (entre tabs del bottom nav): opacity 0 → 1, 0.15s
